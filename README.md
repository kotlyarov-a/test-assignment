
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Установка

```bash
$ npm install
```

#Прописать параметры подклюения к БД в файле

```
$ src/config/appConfig.ts
```

## Запуск приложенрия

```bash
# development
$ npm run start

```

## Запуск тестов

```bash
# e2e tests
$ npm run test:e2e
```
